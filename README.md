# OAuth2Demo
A Spring Boot based OAuth 2.0 Demo

# Start server
```
./gradlew -b demo-server/build.gradle bootRun
```

# Start client
```
./gradlew -b demo-client/build.gradle bootRun
```

# Start resource
```
./gradlew -b demo-resource/build.gradle bootRun
```

# Open in browser
```
http://localhost:8080
```
